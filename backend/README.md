# LocalSkill Exchange - Backend API

Backend API server for LocalSkill Exchange built with Node.js, Express, and MongoDB.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create a `.env` file (see `.env.example` for reference)

3. Start the development server:
```bash
npm run dev
```

The server will run on `http://localhost:5000`
